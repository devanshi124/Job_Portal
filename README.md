# Job Portal

A modern full-stack recruitment platform built with the MERN stack, designed to streamline the hiring process by connecting job seekers with recruiters through an intuitive and scalable web application.

## Overview

Job Portal provides a centralized platform where candidates can explore job opportunities, manage applications, and maintain professional profiles, while recruiters can create job postings, review applications, and efficiently manage the recruitment workflow.

## Key Features

### Candidate Features

* Secure user authentication and authorization
* Create and manage professional profiles
* Browse and search job opportunities
* Apply to jobs seamlessly
* Track application status
* Manage personal account information

### Recruiter Features

* Recruiter-specific authentication and access control
* Create, update, and manage job postings
* View and manage candidate applications
* Access applicant information and profiles
* Streamlined recruitment workflow management

### Authentication & Security

* JWT-based authentication
* Role-based access control
* Protected routes and secure API endpoints
* Password encryption using bcrypt

### State Management

* Centralized state management using Redux Toolkit
* Efficient user session handling
* Optimized application performance and user experience

## Architecture

The application follows a client-server architecture:

* **Frontend:** Handles user interface, routing, and state management.
* **Backend:** Provides RESTful APIs and business logic.
* **Database:** Stores user, recruiter, job, and application data.
* **Authentication Layer:** Ensures secure access and authorization across the platform.
